find_beam_disp_angles¶

CryoNIRSP_Pipeline.Align.Align3Calibration.find_beam_disp_angles(fitsData, extra_clean=False, debug=False)[source]¶

Measure the angle between the dispersion and detector axes for each beam

This is done with pinhole image data and uses a pretty straight-forward Hough Transform to identify the strongest angle.

Parameters
fitsDataData.FitsData

Object containing a single pinhole frame for BOTH BEAMS.

extra_cleanbool

If True then an additional, expensive cleaning step will be done before performing the Hough Transform. This is very good at getting rid of salty bad-pixel data, but in practice this type of noise probably doesn’t affect the Hough Transform too much.

debugbool

If True then various debug images and plots will be produced. The code will pause between beams so that these results can be examined.

Returns
numpy.ndarray

Vector of two numbers; the angle (in degrees) between the dispersion and detector axes for each beam, respectively.
