RampData¶

class CryoNIRSP_Pipeline.common.Data.RampData(file_list)[source]¶

Bases: object

A container for the loading and manipulating of raw DHS data from IR cameras. It is similar to FitsData in how it stores/loads data from FITS files, but it is much simpler (no _DataList) because it has one, specific low-level function.

In addition to loading the FITS files that make up a ramp this class also supports some basic verification and reduction methods that are needed for linearity correction.

Initialize properties, load files, verify the data, and construct auxiliary data.

Parameters
file_listlist

A list of all files (ndr’s) associated with the ramp. These must be in read-order.

correct_camera_specifics()[source]¶

Identify the camera and mode used to collect the data and perform specific pre-processing steps, if needed.

The camera is used to determine the gain.

Fast mode data have the first ndr removed.

Slow mode data are inverted.

load_from_filelist(file_list)[source]¶

Open and load each file in the given file list

There is also a simple check to make sure the file list is in ndr/read order.

All of the data arrays are loaded into a single (N, x, y) numpy array and the corresponding headers are in a single, length N, array.
