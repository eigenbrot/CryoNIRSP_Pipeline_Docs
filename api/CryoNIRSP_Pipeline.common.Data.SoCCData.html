SoCCData¶

class CryoNIRSP_Pipeline.common.Data.SoCCData(data_dir=None, suffix='*.fits')[source]¶

Bases: object

SoCCData performs 2 functions: First, it separates and sorts Calibration Sequence (CS) steps found among raw data in a directory. Secondly, it can initialize and produce a CSStepData object at a requested CS step.

Initialize properties and maybe scan a directory for CS steps.

Parameters
data_dirstr

Path to a directory that will be searched for CS steps

suffixstr

File mask. The full mask will be data_dir/*SUFFIX
