CSStepData¶

class CryoNIRSP_Pipeline.common.Data.CSStepData(CS_step=0, memmap=None)[source]¶

Bases: CryoNIRSP_Pipeline.common.Data.FitsData

CSStepData is an extension of FitsData that loads a single Calibration Curve from a directory containing a full Set of Calibration Curves (SoCC). The user is able to specify which step of the Calibration Sequence (CS) will be loaded and after loading this object behaves exactly like FitsData.

Initialize the object and maybe load some data.

Parameters
data_dirstr

Path to directory in which to look for data

CS_stepint

Only files/data pertaining to this Calibration Sequence step will be loaded

suffixstr

Optional file mask. The full mask is DATA_DIR/*SUFFIX. Defaults to “.fits”

memmapbool

The decision on whether or not to use astropy.io.fits’ memmap feature. If set to ‘None’ then astropy will figure it out for you (recommended).
