DataSetData¶

class CryoNIRSP_Pipeline.common.Data.DataSetData(data_dir=None, suffix='')[source]¶

Bases: object

DataSetData is designed to handle the aggregation of a bunch of files that belong to mulitple slit positions of the Cryo SP instrument. This class reads all files in a directory and populates a dictionary that maps spatial position to filename. It can then create and return instances of the SlitPositionData based on this dictionary. In this way the entire directory only needs to be read once, which saves a lot of time.

Initialize a few parameters and read the directory, if specified.

Parameters
data_dirstr

Path to directory in which to look for data

suffixstr

Optional file mask. The full mask is DATA_DIR/*SUFFIX.
