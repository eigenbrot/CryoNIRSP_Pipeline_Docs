Align3Data([data_dir, memmap])

Align3Data is an extension of FitData designed to only hold data associated with Align3. It inherits all of the functionality of FitsData and defines an extra method that loads all data from a directory who’s headers match the following: Key Value — —– DKIST004 Alignment CRSP_002 Align3.
